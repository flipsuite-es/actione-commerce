## 1. Concepto de marca

**Oucy Studios** = joyas doradas de acero, **elegantes, atemporales y pensadas para durar**. Piezas que
parecen de joyería, que no se oxidan ni manchan la piel, para llevar cada día y para regalar — sin gastar
una fortuna.

- **Posicionamiento:** elegancia atemporal asequible (quiet luxury), no y2k ni bazar.
- **Público:** mujer 25-45 que se autorregala y regala; quiere piezas elegantes de diario que duren.
- **El wedge:** estética curada + historia + comunidad + calidad honesta (el "no se oxida" es **garantía**,
  no eslogan — `docs/16`).

## 8. Reglas de ficha de producto (honestas — protegen pasarela y legalidad)

Para cada pieza: **título claro + material honesto + medida + cuidado**. Ejemplo:

> **Anillo Perla Dorado · Oucy Studios**
> Acero inoxidable con baño dorado y perla sintética. No se oxida, resiste agua y no mancha la piel.
> Ajustable / talla ___. Apto para piel sensible.
> *Cuidado: guárdalo seco; evita perfume directo.*

- Perlas = **"sintéticas / de cristal"**, nunca "naturales".
- **"Acero dorado / baño de oro"**, nunca "oro".
- Herringbone: añade "no doblar en exceso".
